# Change Log

All notable changes to the "rdf-preview" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.3]

### Added

- Highlight nodes and links on hover in the RDF graph preview panel.
- Search functionality in the RDF graph preview panel to find specific nodes and links.

### Fixed

- Fix an issue where the graph will keep fit to screen after dragging nodes around in the RDF graph preview panel.

## [0.2.1]

### Fixed

- Fix issue where RDF graph preview would not be display.
- Fix a bug causing missing styles in the RDF graph preview panel.

## [0.2.0]

### Added

- Hot reloading of the RDF graph preview when the source file is modified.
- Add reload button to the RDF graph preview panel.
- Add fit to screen button to the RDF graph preview panel.
- Contribution guidelines in `CONTRIBUTION.md`

## [0.1.0]: initial release

### Added

- Initial release of RDF-Preview extension for VS Code.
